# 安装和使用说明

## 安装步骤

### 1. 准备图标文件

在 `images/` 目录下放置以下图标文件：
- `icon16.png` (16x16像素)
- `icon48.png` (48x48像素)
- `icon128.png` (128x128像素)

如果没有图标，可以从原项目复制，或使用在线图标生成器创建。

### 2. 加载扩展

1. 打开Chrome浏览器
2. 在地址栏输入 `chrome://extensions/` 并回车
3. 开启右上角的"开发者模式"开关
4. 点击"加载已解压的扩展程序"按钮
5. 选择 `gofullpage-local` 文件夹
6. 扩展加载完成后，会在浏览器工具栏显示图标

### 3. 使用扩展

1. 访问任意网页
2. 点击浏览器工具栏中的扩展图标
3. 点击"开始截图"按钮
4. 等待截图完成（可能需要几秒到几十秒，取决于页面大小）
5. 截图完成后会自动打开结果页面

## 功能说明

### 截图功能
- 自动滚动页面并捕获所有内容
- 支持长页面和宽页面
- 自动拼接多个截图区域

### 编辑功能
- **裁剪**：选择并裁剪图片区域
- **绘制**：在图片上自由绘制
- **文字**：添加文字标注
- **箭头**：绘制箭头指向
- **矩形**：绘制矩形框
- **撤销/重做**：支持操作历史

### 下载功能
- 支持PNG、JPEG、WebP格式
- 可下载为PDF（需要额外库支持）
- 自动保存到浏览器下载文件夹

### 历史记录
- 自动保存所有截图
- 按时间排序显示
- 支持删除单个或全部截图

## 注意事项

1. **受限页面**：以下页面无法使用扩展：
   - `chrome://` 页面
   - `chrome-extension://` 页面
   - `edge://` 页面
   - `about:` 页面

2. **性能**：大型网页截图可能需要较长时间，请耐心等待

3. **存储**：所有截图保存在浏览器本地存储中，不会上传到任何服务器

4. **隐私**：完全本地处理，不收集任何数据，不连接任何远程服务器

## 故障排除

### 截图失败
- 检查是否在受限页面
- 尝试刷新页面后重试
- 检查浏览器控制台是否有错误信息

### 图标不显示
- 确保 `images/` 目录下有图标文件
- 检查文件名是否正确（icon16.png, icon48.png, icon128.png）

### 扩展无法加载
- 确保开启了"开发者模式"
- 检查 `manifest.json` 文件是否正确
- 查看扩展管理页面的错误信息

## 技术限制

由于Chrome扩展API的限制，当前实现使用以下方法：
- 通过滚动页面并多次调用 `chrome.tabs.captureVisibleTab` API
- 将多个截图拼接成完整图片
- 这种方法对某些动态内容可能不够完美
