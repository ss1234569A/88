# 图标文件说明

请在此目录下放置以下图标文件：

- `icon16.png` - 16x16像素的图标
- `icon48.png` - 48x48像素的图标  
- `icon128.png` - 128x128像素的图标

## 图标要求

- 格式：PNG
- 尺寸：16x16, 48x48, 128x128
- 背景：透明或白色
- 建议使用相机或截图相关的图标

## 临时解决方案

如果没有图标文件，可以：
1. 使用在线图标生成器创建
2. 从原项目中复制图标文件
3. 使用简单的占位图标

## 图标建议

可以使用以下免费图标资源：
- Flaticon (https://www.flaticon.com)
- Icons8 (https://icons8.com)
- Material Icons (https://fonts.google.com/icons)

搜索关键词：camera, screenshot, capture, photo
